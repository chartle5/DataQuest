# NovaCura Analytics — Frontend demo

This folder contains a simple static UI prototype for the NovaCura Analytics healthcare ML demo.

To open locally, simply open `frontend/index.html` in a browser (no server required):

Mac / Linux:

    open frontend/index.html

Windows (PowerShell):

    start frontend\index.html

Notes:
- The backend/data integration will be added later. This UI only collects the selected condition and the requested `top N` value and displays a placeholder message.
